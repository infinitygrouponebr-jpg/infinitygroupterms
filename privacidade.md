O Infinity Bot ("Nós", "Nosso" ou "Bot") respeita a privacidade de seus usuários e é comprometido em proteger os dados que coleta. Esta Política de Privacidade explica como coletamos, usamos e protegemos suas informações ao utilizar nossos serviços no Discord.

1. Dados que Coletamos
Para que o bot funcione corretamente em seu servidor, nós podemos coletar e armazenar temporária ou permanentemente os seguintes dados:

Identificadores Básicos: Seu ID de Usuário do Discord e o ID do Servidor (Guild ID).

Dados de Interação: Saldo de moedas virtuais, níveis de experiência (XP), histórico de apostas e registros de moderação (avisos/banimentos) vinculados ao seu ID.

Conteúdo de Mensagens: Para o funcionamento do Sistema de Tradução, o bot lê o conteúdo das mensagens enviadas nos canais configurados. Nós não armazenamos o texto das suas mensagens em nosso banco de dados. O texto é processado em tempo real, enviado para a API de tradução e imediatamente descartado.

2. Como Usamos os Dados
Para fornecer as funcionalidades principais do bot (tradução dinâmica usando Webhooks, comandos de economia, moderação).

Para manter o controle de saldos, níveis e inventários do sistema de economia.

Para aplicar ações de moderação (logs de segurança do servidor).

3. Compartilhamento com Terceiros
O Infinity Bot não vende, aluga ou compartilha seus dados pessoais com terceiros para fins de marketing.

O conteúdo das mensagens traduzidas é processado via APIs externas de tradução (ex: Google Translator API) de forma anônima e em tempo real.

Transações financeiras reais (compras Premium/VIP) são processadas inteiramente pela plataforma nativa do Discord (via Stripe) ou gateways oficiais. O bot não tem acesso aos dados do seu cartão de crédito.

4. Remoção de Dados
Você tem o direito de solicitar a exclusão dos seus dados do nosso banco de dados (como zerar seu saldo de economia ou apagar seu registro de níveis). Para isso, os administradores do servidor podem remover o bot, ou o usuário pode entrar em contato com a equipe de desenvolvimento.
